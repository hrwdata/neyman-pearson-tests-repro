# Raw Source Files

This directory distributes the OCR text source used by the repository:

- `paper_ocr.txt`: OCR-derived text for the relevant sections, example structure, and published numerical targets.

If you want to compare OCR-derived text against the scanned paper, place your own local copy at:

```text
data/raw/paper_pdf.pdf
```

The scanned PDF is optional local input. It is not required for the default reproduction pipeline and is not bundled with the repository.
